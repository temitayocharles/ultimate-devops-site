What's Inside?

The container ships with:

  • code-server – Remote VS Code in browser
  • Grafana – Dashboards for Prometheus & container stats
  • Prometheus – Metrics scraping
  • Vault – Secrets manager with OIDC-ready bootstrap
  • NGINX Reverse Proxy – HTTPS with `.local` domain mapping
  • Tools: AWS CLI, Terraform, kubectl, helm, zsh, stern, glab

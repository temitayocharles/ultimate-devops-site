🚀 Getting Started

Clone the GitHub repo and run:

docker compose up -d

Access services via:

  • https://code.local
  • https://grafana.local
  • https://vault.local

🔐 TLS Setup (Self-Signed)

Using mkcert and NGINX reverse proxy, all *.local domains are secured. No setup needed—just run and access.

📂 Persisting Volumes

Mount your external volume to /data or /vault-backup. Example:

-v /Volumes/256-B:/vault-backup

📎 CLI Tools Installed

  • Docker, Compose, AWS CLI
  • Terraform, kubectl, Helm
  • Trivy, Checkov, SOPS
  • Vault CLI, glab, yq, jq, stern
